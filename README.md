# WannOfficial — Portfolio

Bot Telegram & Automation Developer portfolio built with Next.js 14, React 18, and Tailwind CSS.

## Tech Stack

- **Next.js 14** — React framework with App Router
- **React 18** — UI library
- **Tailwind CSS 3** — Utility-first CSS framework
- **Font Awesome 6** — Icons
- **JetBrains Mono** — Monospace font

## How to Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Deploy on Vercel

Connect this repo to [Vercel](https://vercel.com) — it auto-detects Next.js, no config needed.
